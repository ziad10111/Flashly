# Active Recall Study Notes

Active recall is a study method where learners try to retrieve an answer from memory before checking the source. This makes practice more effective than rereading because it reveals what the learner can explain without support.

Spaced review means revisiting material over time instead of practicing everything once. Short review sessions spread across several days can help important ideas stay available for future use.

A useful flashcard usually asks one clear question and expects one focused answer. If a topic has several facts, it is better to create several smaller cards than one overloaded card.

Progress tracking helps learners notice patterns. Cards answered correctly can be reviewed less often, while missed cards should return sooner for more practice.
